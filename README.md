# DIG Cheat Sheet

### Ex: dig google.com
```dig [hostname]```

### Ex: dig google.com TXT
```dig [hostname] [record type]```

### Ex: dig google.com  +short
```dig [hostname] +short```

### Ex: dig @google.com
```dig @[nameserver address] [hostname]```

### Ex: dig google.com +trace
```dig [hostname] +trace```

### Ex: dig -X 142.251.128.78
```dig -X [IP address]```

### Ex: dig google.com any
```dig [hostname] any```
