# DIG Cheat Sheet

```dig [hostname]```
Ex: dig google.com

```dig [hostname] [record type]```
Ex: dig google.com

```dig [hostname] +short```
Ex: dig google.com

```dig @[nameserver address] [hostname]```
Ex: dig @google.com

```dig [hostname] +trace```
Ex: dig google.com +trace

```dig -X [IP address]```
Ex: dig -X 142.251.128.78

```dig [hostname] any```
Ex: dig google.com any
